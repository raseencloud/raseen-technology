# Folder Responsibilities

## Root

Contains the public website and essential project files.

---

## css/

Contains all Cascading Style Sheets.

---

## js/

Contains all JavaScript files.

---

## images/

Contains optimized images (WebP / AVIF).

---

## icons/

Contains SVG icons only.

---

## fonts/

Contains local fonts used by the website.

---

## docs/

Contains all official project documentation.

---

## governance/

Contains governance, compliance, legal, ethics, and risk management documentation.

- Follow a modular architecture.
